# MinimalFetch
This is a command line System Information tool written in pure C++ in just 104 SLOC. 

Installation :
```
g++ minimalfetch.cpp -o minimalfetch
sudo cp minimalfetch /usr/bin/minimalfetch
```
Tested in :
```
Gentoo Linux
Arch Linux
```
